Detailed Description

Geometrical FLIRT Phrases (GFP) for matching 2D laser scans represented FLIRT words.

It includes methods for building the search index and matching

"Geometrical FLIRT Phrases for Large Scale Place Recognition in 2D Range Data", G. D. Tipaldi, L. Spinello, W. Burgard -- Int. Conf. Robotics and Automation (ICRA) 2013

Author:
Luciano Spinello

void gefp_engine::build_tfidf ( void  )

Builds TF-IDF index for standard and weak verification matching methods.

It implements IDF, TF, TF-IDF, wordcount and improved TF-IDF models proposed in: Gerard Salton and Christopher Buckley: "Term-weighting approaches in automatic text retrieval", Information processing & management, vol. 24, no. 5, 1988, Elsevier

void gefp_engine::prepare ( void  )

Prepares indeces and cache for matching.

Executed once at the beginning. Builds TF-IDF index for the dataset and norms all vectors on the dataset. Allocates also needed memory. Optionally it generates bag-of-distances
